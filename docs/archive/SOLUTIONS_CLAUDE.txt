Dashboard Design Brief: Hyper-Professional Modern Interface
Project Overview
Create an extensively detailed, production-ready dashboard with cutting-edge visual design. Target aesthetic: premium SaaS platforms like Linear, Vercel, and Stripe. Every pixel matters.

Core Design Philosophy
"Elevated Minimalism" - Sophisticated simplicity with thoughtful micro-details that reveal themselves through interaction. Think: expensive, refined, intelligent.

‚ú® Micro-Interactions (Critical for Premium Feel)
Hover States (Apply Everywhere)

Scale: 1.02 for cards, 1.05 for buttons
Duration: 250ms with ease-out
Brightness: filter: brightness(1.05) for colored elements

Loading States

Skeleton screens with shimmer animation
Fade-in for loaded content (opacity + transform)
Stagger animations for lists (delay: calc(var(--index) * 50ms))

‚ö° Performance Rules

Use transform and opacity for animations (GPU-accelerated)
Apply will-change ONLY during active animations
Implement content-visibility: auto for long lists
Use loading="lazy" for all images below fold
Minimize layout shifts with fixed aspect ratios

css.heavy-component {
  content-visibility: auto;
  contain-intrinsic-size: 500px;
}
